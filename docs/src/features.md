# Features

- [Lookup registrations](https://github.com/socrata/diceberg/blob/main/src/cli/info/lookup/registration.rs)
- [Lookup inventories](https://github.com/socrata/diceberg/blob/main/src/cli/info/lookup/inventory.rs)
- [Get schema of table](https://github.com/socrata/diceberg/blob/main/src/cli/info/table/schema.rs)
- [Execute sql against table](https://github.com/socrata/diceberg/blob/main/src/cli/sql.rs)
- [CLI that exposes everything through sub commands](https://github.com/socrata/diceberg/blob/main/src/cli/mod.rs)
- Lib with table/sql api, catalog/context access
- [Table manifest size, table data size](https://github.com/socrata/diceberg/blob/main/src/cli/info/table/stats)
- [Table partitions](https://github.com/socrata/diceberg/blob/main/src/cli/info/table/partition.rs)
- [Snapshot history, snapshot details](https://github.com/socrata/diceberg/blob/main/src/cli/info/table/history)
- [JSON or IPC(arrow format) sql output format](https://github.com/socrata/diceberg/blob/main/src/cli/sql.rs)
- [IPC print utility](https://github.com/socrata/diceberg/blob/main/src/cli/util/ipc/print.rs)
- [Execute sql against dataframes(IPC) input to stdin](https://github.com/socrata/diceberg/blob/main/src/cli/util/ipc/query.rs)
- [MCP server](https://github.com/socrata/diceberg/blob/main/src/cli/serve/mcp.rs)
- [Embedded local LLM chat with Ratatui terminal UI](https://github.com/socrata/diceberg/blob/main/src/cli/ai.rs)
